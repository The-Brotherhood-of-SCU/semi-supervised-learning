# 半监督AE模型简介

执行多任务学习。

```
            ->Decoder      =>图像重建
Encoder->低维表示
            ->Classifier   =>分类
```

对于分类任务，使用交叉熵损失函数。
对于图像重建任务，使用均方误差损失函数。

对于有标记的样本，两者产生的损失均可训练。
对于无标记的样本，仅使用图像重建任务的损失进行训练。
